[[최대공약수]] 의 확장.
[[유클리드 알고리즘]]을 사용하는 과정에서 나오는 값을 이용해 구현한다

```cpp
int xgcd(int a, int b, int& x, int& y) {
// 앞 두단계만 필요하므로
// 이를 두개의 [3] 크기 배열로 표현
	int now[3] = {1, 0, a};
	int next[3] = {0, 1, b};
	while (next[2] > 0) {
		int q = now[2] / next[2];
		// GCD 연산 시작
		// 지금 단계에서 다음단계 빼면->다다음 단계 (S_i+2 = S_i - q*S_i+1)
		for (int i = 0; i < 3; i++) now[i] -= q * next[i];
		// 순서 정렬
		swap(now, next);
	}
	x = now[0], y = now[1];
	return now[2];
}
```
