여러 개의 합동식이 모듈로 관계가 서로소이면
합동식의 해를 각 모듈로의 곱들 안에서 "유일하게" 찾을 수 있다

$x \equiv2$ $mod$ $3$
$x \equiv 3$ $mod$ $5$
$x \equiv 2$ $mod$ $7$

이 해는 x = 23 으로
3 * 5 * 7 까지의 수 중 유일한 해이다

-> $M$을 $m_1, m_2,..., m_k$ 로

$A = (\sum_{i = 1}^ka_ic_i)(mod$ $M)$

A mod M 을 구하는 "빠른" 알고리즘
~~~psudo
//M[i] = M / m[i]
sum = 0;
for i in k
	a[i] = A mod m[i]
	c[i] = M[i]*(M[i]^-1 mod mi)
	sum = sum+ a[i]*c[i]
	if(sum>=M) sum -= M
~~~
